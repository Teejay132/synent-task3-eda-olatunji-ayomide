# Netflix Exploratory Data Analysis (EDA)

# Problem Statement
Identify trends and patterns in the Netflix dataset.

# Dataset
Netflix Titles Dataset - sourced from Kaggle

# Approach
- Summary statistics on content type, country and ratings
- Trend analysis of content added per year
- Correlation between release year and year added
- Top 10 genres analysis

# Results
Netflix grew rapidly after 2015, is dominated by Movies,
and mostly features International and Drama content
